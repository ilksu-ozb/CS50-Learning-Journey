# Problem Set 8: Homepage (Introduction to Web Development)

## Problem Overview
The objective was to build a personal website using HTML, CSS, and JavaScript. This project required creating a multi-page site that is not only functional but also responsive and visually coherent. It served as my first deep dive into how the "front-end" of the internet works.

## Key Concepts Learned
* **HTML:** Structuring content with semantic tags like `<header>`, `<nav>`, and `<section>`.
* **CSS:** Styling pages using Flexbox and Grid for layout, and implementing responsive design with Media Queries.
* **JavaScript:** Adding interactivity, such as button clicks, alerts, or dynamic content updates.
* **Bootstrap:** Utilizing a front-end framework to quickly build a polished and mobile-friendly interface.

## Design Thinking
When building my homepage, I focused on:
1. **User Experience (UX):** Creating a simple navigation bar that works on both desktop and mobile.
2. **Visual Consistency:** Using a consistent color palette and typography across all pages.
3. **Interactivity:** Adding a JavaScript-powered feature (like a quiz or a dynamic greeting) to engage the user.
